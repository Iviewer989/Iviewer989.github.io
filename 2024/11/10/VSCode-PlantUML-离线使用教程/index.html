VSCode PlantUML 离线使用教程

Posted on 2024-11-10 Views:

目录结构

plantuml_portable/
├── .vscode/ # VSCode 配置目录
├── Graphviz-12.2.0-win64/ # Graphviz 可执行文件
├── jdk8u432-b06-jre/ # Java 运行环境
├── diag.puml # PlantUML 示例文件
├── jebbs.plantuml-2.18.1.vsix # PlantUML 插件安装包
├── plantuml.jar # PlantUML JAR 文件
└── README.md # 说明文档

安装步骤

1. 安装 VSCode PlantUML 插件

  1. 打开 VSCode
  2. 按 Ctrl+Shift+P 打开命令面板
  3. 输入 “Install from VSIX”
  4. 选择 jebbs.plantuml-2.18.1.vsix 文件进行安装

2. 配置 PlantUML 插件

  1. 在 VSCode 中按 Ctrl+, 打开设置
  2. 点击右上角的 “打开设置(JSON)” 图标

使用说明

基本操作

  1. 创建或打开 .puml 文件
  2. 使用 Alt+D 预览图表
  3. 使用 Ctrl+Shift+P 然后输入 “PlantUML: Export Current File Diagrams” 导出图表

  2. 类图 (Class Diagram)
  3. 用例图 (Use Case Diagram)
  4. 活动图 (Activity Diagram)
  5. 状态图 (State Diagram)

故障排除

常见问题

  1. 无法预览图表
    • 检查 Java 路径配置
    • 检查 plantuml.jar 路径配置
    • 检查 Graphviz dot 路径配置
  2. 图表渲染错误
    • 检查 PlantUML 语法
    • 确保文件以 @startuml 开始，@enduml 结束

快捷键参考

  • Alt+D: 预览图表
  • Ctrl+Shift+P: 打开命令面板
  • Ctrl+S: 保存文件并刷新预览

实用链接

  • PlantUML 官方文档
  • PlantUML 语言参考指南

注意事项

  1. 路径中不要使用中文
  2. 路径使用正斜杠(/)或双反斜杠(\)
  3. 确保所有依赖文件都在正确位置
  4. 建议使用相对路径配置

维护说明

  1. 定期更新 PlantUML JAR 文件
  2. 保持 VSCode 插件更新
  3. 备份配置文件

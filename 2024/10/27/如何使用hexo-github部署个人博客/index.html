十一、常见问题解答

1. 部署后页面没有更新？

  • 确保已经执行了 hexo clean、hexo g、hexo d。
  • 检查 _config.yml 中的 url 和 deploy 配置是否正确。

2. 部署时报错或失败？

  • 确认部署插件已正确安装。
  • 检查 Git 仓库地址和分支配置。
  • 如果是权限问题，考虑使用 SSH 密钥进行身份验证。

3. 代码高亮不起作用？

  • 检查主题的代码高亮配置是否正确。
  • 确保在代码块中正确指定了语言。

十二、总结

通过以上步骤，您可以使用 Hexo 和 GitHub Pages 快速搭建一个个人博客。相比其他方法，Hexo 对于初学者来说更为友好，拥有丰富的中文文档和社区支持。

优点：

  • 简单易用：Hexo 的安装和使用相对简单，配置方便。
  • 丰富的主题和插件：拥有大量美观的主题和实用的插件。
  • 高速生成：Hexo 生成静态页面速度快，适合博客部署。
